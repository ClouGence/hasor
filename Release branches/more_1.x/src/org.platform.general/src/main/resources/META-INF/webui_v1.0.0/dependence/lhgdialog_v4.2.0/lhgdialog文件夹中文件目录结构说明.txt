【文件&目录】
lhgdialog.js - 未压缩的lhgDialog弹出窗口组件代码源文件
lhgdialog.min.js - 压缩的jQuery版的lhgDialog弹出窗口组件代码
lhgcore.lhgdialog.min.js - 压缩的独立版本的lhgDialog弹出窗口组件代码
[skins] 文件夹 - lhgDialog弹出窗口组件的皮肤文件夹，注意此文件夹一定要和组件代码文件放到同一目录下
